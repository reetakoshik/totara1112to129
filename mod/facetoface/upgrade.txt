This files describes API changes in /mod/facetoface/*,
information provided here is intended especially for developers.

=== 12.7 ===

* signup\condition\user_is_enrolable::pass() has been updated to allow suspended users to be enrolled into seminar event
* mod_facetoface_renderer::attendees_export_form() has been changed to attach an aria-label attribute to the <select> element
* mod_facetoface_renderer::display_bulk_actions_picker has been changed to attach a <label> element to a form field

=== 12.6 ===

* seminar_event_list::form_seminar() has been renamed to seminar_event_list::from_seminar()
* mod_facetoface_renderer::print_userlist_table() has been updated to format user idnumbers before output

=== 12.3 ===

* mod_facetoface_renderer::render_seminar_event has been changed to prevent a record not found exception if a user has a deleted job assignment
